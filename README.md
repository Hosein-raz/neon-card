## 📖 About The Project

The **Neon Glassmorphism Cards** project is a highly engaging, modern UI component that brilliantly fuses two popular web design trends: Neon Glowing effects and Glassmorphism. Built entirely with pure HTML and CSS, these cards feature vibrant, skewed gradient backgrounds that dynamically expand and un-skew on hover, revealing floating frosted-glass elements. It is an exceptional showcase of advanced CSS animations and interactive design.

## ✨ Key Technical Features

This project relies on zero JavaScript and focuses entirely on advanced CSS rendering techniques:

* **Dynamic Skew Animations:** Utilizes `transform: skewX(15deg)` on the vibrant background elements, smoothly transitioning to `0deg` upon hover to create a bold, unfolding mechanical effect.
* **Complex Glassmorphism:** The main content box and the floating micro-elements employ `backdrop-filter: blur(10px)` with semi-transparent `rgba(255,255,255,0.05)` backgrounds to achieve a highly realistic frosted glass texture.
* **Pure CSS Glow Effects:** Achieves the intense neon glow by applying `filter: blur(30px)` to the `::after` pseudo-elements of the cards, utilizing multiple distinct `linear-gradient` color schemes.
* **Floating Micro-Interactions:** Custom `@keyframes` (`translateY`) animate the tiny glass squares that appear precisely on hover, adding a dynamic, 3D spatial feel to the dark-themed environment (`#1d061a`).

## 🚀 How to Use

Integrating these neon glass cards into your own dark-themed project is simple:

1. Clone the repository:
```bash
git clone https://github.com/Hosein-raz/neon-card.git
```
2. Copy the HTML structure from `index.html`.
3. Import the styling from `style.css` into your project.
(Note: You can easily change the neon colors by modifying the `linear-gradient` hex codes in the CSS pseudo-elements).

## 📂 Project Structure

```
neon-card/
├── index.html    # The semantic structure of the cards
└── style.css     # Dark theme layout, Glassmorphism, and hover animations
```
